A viris on a mission...

Techidemic was a fun five week project for TC 455 based on the theme "destruction". We wanted to make a fast paced game with a rotation-based character controller. We succeeded in doing both thanks to some good programming, fast paced music, and reflex-based level designs.

My work on the project was designing level one and setting up prefabs for the different obstacle types. I also worked on the scoring system that rewards players for going fast, hitting the objectives, and avoiding obstacles. This was difficult because some players would just go as fast as possible and still achieve a good score. It was a unique challange finding an equation that would take all gameplay features into account.
